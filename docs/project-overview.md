# Project Overview

This project demonstrates how Python can clean and validate messy CSV data before it is used for reports, dashboards, imports, or automation workflows.

## Problem

CSV files exported from CRMs, spreadsheets, e-commerce systems, and internal tools are often messy.

Common problems include:

- Extra spaces in column names
- Missing values
- Duplicate rows
- Empty rows
- Inconsistent formatting
- Required columns missing from the file

## Solution

This project uses Python and pandas to:

1. Read the CSV file
2. Normalize column names
3. Trim text values
4. Remove empty rows
5. Remove duplicates
6. Validate required columns
7. Count missing values
8. Export a clean CSV file
